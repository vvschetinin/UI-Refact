# 🧱 Адаптивная CSS-сетка: Документация

## 📌 Общее описание

Это **гибкая и полностью адаптивная сетка** для позиционирования элементов на сайте.  
Реализована с использованием **Flexbox**, но также может быть адаптирована под **CSS Grid**.

Позволяет точно контролировать ширину, расположение и поведение блоков на экранах разных размеров — от мобильных телефонов до больших дисплеев.

---

## 🔁 Брейкпоинты (точки останова)

Сетка поддерживает **6 уровней адаптации**:

| Брейкпоинт | Ширина экрана | Описание                             |
| ---------- | ------------- | ------------------------------------ |
| `xs`       | 0px           | Мобильные устройства (по умолчанию)  |
| `sm`       | 550px         | Планшеты в портретном режиме         |
| `md`       | 800px         | Планшеты в альбомном режиме          |
| `lg`       | 1100px        | Настольные мониторы среднего размера |
| `xl`       | 1400px        | Большие мониторы                     |
| `xxl`      | 1800px        | Очень большие экраны (4K и выше)     |

### Пример использования:

```html
<div class="col-xs-100 col-sm-50 col-md-33 col-lg-25 col-xl-20 col-xxl-16"></div>
```

---

## 📐 Ширина колонок

Сетка поддерживает **ширины от 1% до 100% с шагом 1%**, что даёт очень высокую гибкость при верстке.

### Примеры классов:

```html
<div class="col-10">Ширина 10%</div>
<div class="col-50">Ширина 50%</div>
<div class="col-100">Полная ширина</div>
```

А также адаптивные:

```html
<div class="col-xs-100 col-md-50 col-lg-25">Адаптивный блок</div>
```

---

## 🛠️ Функционал и возможности

### 1. **Гибкие контейнеры**

Используются классы `.row` и `.col`, чтобы создавать строки и колонки:

```html
<div class="row">
  <div class="col-30">Блок A</div>
  <div class="col-70">Блок B</div>
</div>
```

### 2. **Поддержка отступов (гуттеров)**

Все элементы имеют внутренние отступы (`padding`) и внешние отступы у `.row`, чтобы избежать вылезания контента за границы.

### 3. **Выравнивание и распределение**

Доступны классы для управления выравниванием:

| Класс              | Действие                        |
| ------------------ | ------------------------------- |
| `.justify-start`   | Выравнивание по левому краю     |
| `.justify-center`  | Центрирование                   |
| `.justify-end`     | По правому краю                 |
| `.justify-between` | Растяжка между элементами       |
| `.justify-around`  | Равные отступы вокруг элементов |
| `.align-center`    | Вертикальное центрирование      |

Пример:

```html
<div class="row justify-center align-center">
  <div class="col-50 text-center">Центральный блок</div>
</div>
```

### 4. **Вспомогательные классы**

- Отступы: `.mt-10`, `.mb-10`, `.ml-10`, `.mr-10`
- Паддинги: `.pt-10`, `.pb-10`, `.pl-10`, `.pr-10`
- Выравнивание текста: `.text-left`, `.text-center`, `.text-right`

## 🔄 Расширяемость

Если потребуется, можно легко расширить функционал:

| Что можно добавить               | Описание                              |
| -------------------------------- | ------------------------------------- |
| Offset (сдвиг колонок)           | Например: `.offset-md-20`             |
| Push/Pull (перестановка колонок) | Например: `.push-lg-50`               |
| Скрытие/показ элементов          | Например: `.hidden-sm`, `.visible-md` |
| Поддержка CSS Grid               | Для более сложных макетов             |
| Темизация через CSS переменные   | Для dark mode или темизации           |
